#EliteFootballAI

EliteFootballAI is a full-stack web application that provides real-time football scores, live commentary, and AI-powered match predictions. It features a seamless intergration with the  Safaricom Daraja API for MPESA STK Push payments and utilizes OpenAI's GPT models to generate dynamic marketing copy and predictions.

##Features
• *Live Scores & Commentary:*
Real-time updates for ongoing matches.
• *AI Predictions:* Automated VIP and Jackpot tips generated via OpenAI.
• *MPESA Intergration:* Secure STK Push payment processing via safaricom Daraja API.
• *AI Ad Generator:* Tools for admins to generate creative ad copy using AI.
• *Automated Tasks:* Cron  jobs to refresh predictions every 5 minutes.

## Tech Stack
•*Frontend:* HTML5, CSS3, Vanilla Javascript.
•*Backend:* Node.js, Express.js.
•*APIs:* Safaricom Daraja(M-pesa),
OpenAI API.
•*Tools:* Axios, Node-cron, Dotenv.
